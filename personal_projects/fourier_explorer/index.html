Fourier Series Visualization (C++)

Dec 2021

WASM Build!

The fourier series visualization allows the user to draw out a discrete path, and then observe a visualization of a finite fourier series approximating the path. Each term in the fourier series is represented as an arrow- it's length the magnitude of the terms coefficient, the speed and direction it rotates it's frequency, and the initial direction it's phase.


[click here to run the 2d fourier series visualization]

The image demo performs a 2d discrete fourier transform on an image, which for the web assembly build has been hard coded to a lizard.
The left image shows the original image. The middle shows the noramlized log of the image in the frequency domain. The right shows the reconstruction of the image from the frequency domain.
The demo allows you to try removing frequencies and observing the effect on the reconstruction.


[click here to run the image transform demo]

Path Demo Controls

Press Z to clear any active visualization. Then click repeatedly to draw a path on the canvas. Press 'SPACE' to close the path, and re-enter visualization mode.

Initially the series will have only 3 terms so it will not be very accurate. Press 'P' to increase the number of terms. Press 'O' to reduce the number of terms.
You can pan the camera by clicking and holding the left mouse button. The scroll wheel can be used to zoom in or zoom out.
'F' will toggle making the camera follow the current position of the fourier series.

Image Transform Demo Controls

Click repeadetly in the frequency domain visualization (middle image) to trace out a closed path. Then press 'I' to remove all frequencies outside this path, or 'O' to remove all frequencies inside the path. After pressing either 'I' or 'O', the reconstruction image will be updated. You can then trace out and apply another path, or press the 'R' button to reset.

About

Github project link

There were two main sources of inspiration for this project. The first was this great video by 3Blue1Brown, which I absolutely loved the visualizations in and wanted to try recreating. The second was a computer graphics class I had taken the fall of 2021, which in one lecture briefly discusses transforming images to and from the frequency domain. I thought that was really neat, and I decided to try to write this program over the winter break to try to recreate both of these visualizations.

The image demo uses a Fast Fourier Transform implementation that I wrote.
