About

Github project link

Shortly after I created the 2d physics engine, I came across a developer talk discussing the development of portal and some of the things they did to get the physics to work with that game. I was inspired by this to try to see if I could reproduce this myself, albiet in 2d, using my own physics engine, which was the origin of this project.

The project has a level editor that was used to create all of the stages in the game.

I am pretty proud of how this came out both artistically and technically for my skills at the time, but gameplay wise it is pretty bad. The character controller is physics based, but is not well refined which makes the handling feel terrible, and the puzzle design is also very poor.
