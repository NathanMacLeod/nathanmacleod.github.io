This is a 2d physics engine I wrote my senior year of highschool. One interesting aspect of the engine is that it supports concave objects, as one of my goals with this project was for the user to be able to draw arbitrary objects at runtime. I got this working with a very convoluted and slow collision detection method I came up with, which works but has pretty terrible performance. Correctness wise it works decently though, as can be seen in the video.
